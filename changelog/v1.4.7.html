Version 1.4.7 - Monthly Summary and Fund Allocation Integration

Features

  • Unified Financial View
    • Combined monthly summary and fund allocation into a single card
    • Synchronized date selection across all financial components
    • Enhanced responsive layout for desktop (side-by-side view)
    • Contextual action links in budget mismatch warnings
    • Streamlined user interface for better financial insights

Technical Details

Layout Integration

  • Combined monthly summary and fund allocation into a single card in apps/webapp/src/app/app/page.tsx
  • Updated layout for desktop to show monthly summary and fund allocation side by side
  • Improved mobile view with stacked components and consistent spacing

Date Synchronization

  • Modified MonthlySummaryCard to accept a controlled date from parent component in apps/webapp/src/components/MonthlySummaryCard.tsx
  • Updated SpendingGraph to expose its date selection state to parent components in apps/webapp/src/components/SpendingGraph.tsx
  • Synchronized date selection between the monthly summary and fund allocation in apps/webapp/src/app/app/page.tsx
  • Implemented shared state management to ensure consistent filtering

Budget Warning Enhancement

  • Added contextual action links to the budget mismatch warning in apps/webapp/src/components/MonthlySummaryCard.tsx
  • Added "Add to Savings" link for the transactions option
  • Added "Adjust Budget" link for the budgets option
  • Enhanced warning visibility based on discrepancy severity

Implementation Notes: This update creates a more cohesive financial dashboard by integrating the monthly summary and fund allocation into a unified view. The shared date selection ensures that users are always looking at the same time period across all financial information. The enhanced budget warning now provides actionable links to help users quickly resolve budget discrepancies, either by adding the difference to savings or by adjusting their budget.
