Version 1.2.0 - Expense Sharing

Features

  • Expense Sharing
    • Share transaction data with others via unique links
    • Configurable expiration dates for shared links
    • No authentication required for viewing shared data
    • Shared view includes transactions and category summaries

Technical Details

Backend Implementation

  • Implemented createShareLink mutation in services/backend/convex/sharing.ts
  • Implemented getSharedTransactions query in services/backend/convex/sharing.ts
  • Implemented getSharedCategorySummary query in services/backend/convex/sharing.ts
  • Created shareLink type definitions in services/backend/convex/_generated/dataModel.d.ts
  • Added security checks to verify link validity and expiration

Frontend Implementation

  • Created ShareButton component in apps/webapp/src/components/ShareButton.tsx
  • Created ShareModal component in apps/webapp/src/components/ShareModal.tsx
  • Added sharing capabilities to TransactionList in apps/webapp/src/components/TransactionList.tsx
  • Added sharing capabilities to CategoryPieChart in apps/webapp/src/components/CategoryPieChart.tsx
  • Created shared view page at apps/webapp/src/app/shared/[id]/page.tsx
  • Created SharedTransactionView component in apps/webapp/src/components/SharedTransactionView.tsx

Implementation Notes: The sharing functionality generates unique links that allow viewers to see transaction data and category summary charts without requiring authentication. The share links have configurable expiration dates for security.
