Version 1.2.1 - Share Management Improvements

Features

  • Share Management
    • Permanent share links (no expiration option)
    • Dedicated shares management page
    • Listing of all active share links
    • Ability to delete individual share links
    • Display of share creation date for tracking

Technical Details

Backend Implementation

  • Updated createShareLink to support permanent shares in services/backend/convex/sharing.ts
  • Implemented listUserShares query in services/backend/convex/sharing.ts
  • Implemented deleteShareLink mutation in services/backend/convex/sharing.ts
  • Enhanced data schema to support null expiration dates

Frontend Implementation

  • Updated ShareModal to include "No expiration" option in apps/webapp/src/components/ShareModal.tsx
  • Created SharesListItem component for displaying a share in apps/webapp/src/components/SharesListItem.tsx
  • Created SharesList component for displaying all shares in apps/webapp/src/components/SharesList.tsx
  • Created shared links management page at apps/webapp/src/app/shares/page.tsx
  • Added link to shared links management page in apps/webapp/src/app/app/page.tsx

Implementation Notes: Enhanced share management by adding the ability to create permanent shares (no expiration), view all active shares, and delete existing shares. The share creation date is displayed in the shares list for better tracking. This creates a more complete sharing system for users who want to maintain long-term access to shared financial data.
